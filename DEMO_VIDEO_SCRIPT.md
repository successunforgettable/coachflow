## 📋 Recording Checklist

### Before Recording:
- [ ] Clear browser cache and cookies
- [ ] Prepare test account with clean data
- [ ] Set up screen recording software (Loom, OBS, or similar)
- [ ] Test audio levels
- [ ] Close unnecessary browser tabs
- [ ] Disable notifications
- [ ] Prepare script/notes for narration

### During Recording:
- [ ] Use 1920x1080 resolution
- [ ] Record at 30fps minimum
- [ ] Speak clearly and at moderate pace
- [ ] Pause between scenes for editing
- [ ] Show mouse cursor for clarity
- [ ] Use smooth scrolling and transitions

### After Recording:
- [ ] Edit for pacing and clarity
- [ ] Add background music (optional)
- [ ] Add text overlays for key features
- [ ] Export in HD (1080p)
- [ ] Upload to YouTube/Vimeo
- [ ] Add to landing page

## 4. Source of Truth Setup

**What is Source of Truth?**
Your core business information that powers all generators. Set it once, use it everywhere.

**Fields:**

### Business Information
- **Business Name** - Your company/brand name
- **Target Audience** - Who you serve (demographics, psychographics)
- **Core Offer** - Your main product/service

### Value Proposition
- **Unique Value Proposition** - What makes you different
- **Pain Points** - Problems you solve (3-5 bullet points)
- **Transformation** - Results clients get (before/after)

### Additional Context
- **Industry** - Your market/niche
- **Tone of Voice** - Professional, casual, authoritative, etc.
- **Key Benefits** - Top 3-5 benefits of working with you

**Why It Matters:**
- ✅ Set once, use everywhere
- ✅ Consistent messaging across all content
- ✅ No repetitive form filling
- ✅ All generators auto-populate with this data

**How to Edit:**
1. Click "Source of Truth" in sidebar
2. Update any field
3. Click "Save Changes"
4. All future generations use updated info

## 12. WhatsApp Sequences Generator

**WhatsApp Marketing Sequence Generator**

**What It Creates:**
7-message WhatsApp sequence using Helo.ai's 7-Strategy Framework

**Input Fields:**
- **Service** - Select from dropdown
- **Campaign Goal** - Sales, engagement, event registration, etc.
- **Power Mode Toggle** - Generate 3x more variations

**7-Strategy Framework:**

### Message 1: Hook (Day 1)
- Grab attention with curiosity or value
- Short, punchy, emoji-friendly
- Example: "🔥 Quick question for you..."

### Message 2: Story (Day 2)
- Share relatable story or case study
- Build connection and trust
- Example: "Let me tell you about Sarah..."

### Message 3: Value (Day 3)
- Deliver actionable tip or insight
- Position yourself as expert
- Example: "Here's the #1 mistake I see..."

### Message 4: Social Proof (Day 4)
- Share testimonial or results
- Build credibility
- Example: "Just got this message from a client..."

### Message 5: Offer (Day 5)
- Present your solution
- Clear value proposition
- Example: "I have something that can help..."

### Message 6: Urgency (Day 6)
- Create FOMO or scarcity
- Limited-time bonus
- Example: "Only 3 spots left for..."

### Message 7: Close (Day 7)
- Final call-to-action
- Make it easy to say yes
- Example: "Reply YES if you're interested..."

**Each Message Includes:**
1. **Message Text** - Mobile-optimized length (100-200 chars)
2. **Emoji Suggestions** - Platform-appropriate emojis
3. **Timing** - Best time to send
4. **Expected Response** - What to expect
5. **Follow-up Options** - If they reply/don't reply

**Power Mode:**
- 21-message sequence (3x longer)
- Multiple variations per message
- A/B testing suggestions

**Results Display:**
- **Message-by-Message View** - Navigate through sequence
- **Mobile Preview** - See how it looks on WhatsApp
- **Copy Button** - Copy individual messages
- **Character Count** - Optimal length indicator
- **PDF Export** - Download entire sequence

**Best Practices:**
- ✅ Keep messages short (100-200 characters)
- ✅ Use emojis strategically
- ✅ Send at optimal times (10am-2pm, 7pm-9pm)
- ✅ Personalize with first name
- ✅ Always provide value before asking

**Use Cases:**

## 18. Power Mode Feature

**Power Mode: 3x More Variations**

**What is Power Mode?**
Pro and Agency users can enable Power Mode to generate 3x more content variations in a single generation.

**Available In:**
- Headlines Generator (75 instead of 25)
- HVCO Titles (150 instead of 50)
- Hero Mechanisms (45 instead of 15)
- Ad Copy (45 instead of 15)
- Email Sequences (15 emails instead of 5)
- WhatsApp Sequences (21 messages instead of 7)
- Landing Pages (15 angles instead of 5)
- Offers (15 variations instead of 5)

**How to Enable:**

### Per-Generation Toggle
- Toggle switch on each generator page
- "Power Mode" label with tooltip
- Disabled for Trial users (upgrade prompt)

### Always-On Setting
- Enable in Settings → Power Mode
- Power Mode on by default for all generations
- Can still toggle off per generation

**Benefits:**
- ✅ More creative variations
- ✅ More options to choose from
- ✅ Better A/B testing opportunities
- ✅ Faster content creation
- ✅ Higher quality outputs

**Quota Impact:**
- Uses 1 generation (not 3)
- Same quota cost as regular generation
- More value per generation

**When to Use Power Mode:**
- ✅ Important campaigns
- ✅ A/B testing
- ✅ Client work (agencies)
- ✅ Need many options
- ✅ Exploring different angles

**When to Use Regular Mode:**
- Quick content needs
- Simple variations needed
- Conserving quota (Pro users)
- Testing a single angle

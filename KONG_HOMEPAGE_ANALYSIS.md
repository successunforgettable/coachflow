# Kong Homepage Design Analysis

**URL:** https://getkong.ai

## Key Design Elements Observed

### Visual Style
- **Dark/Light Mode Toggle** - Dual image sets for both themes
- **High-contrast imagery** - Professional screenshots with clear CTAs
- **Emoji usage** - Strategic use of fire emoji (🔥) for emphasis
- **Comparison tables** - "Old Way vs Kong's New AI method" side-by-side
- **Social proof** - Before/After results with data visualization

### Typography & Copy
- **Conversational tone** - Casual, direct, personality-driven
- **Bold claims** - "$7.8 billion in ROAS", "20% ROAS boost guaranteed"
- **Benefit-focused headlines** - Clear value propositions
- **Short paragraphs** - Easy to scan
- **Power words** - "Slash", "Instantly", "Guaranteed"

### Layout Structure
1. Hero section with bold headline + CTA
2. Problem/agitation section
3. Guarantee section with visual
4. Benefits showcase (3 columns)
5. How it works / Method comparison
6. Feature highlights with icons
7. Pricing table
8. FAQ accordion
9. Final CTA

### Call-to-Actions
- Multiple "Sign Up Now" buttons throughout
- Anchor links to pricing (#select-plan)
- Strong final CTA at bottom
